# 🤖 CodeReviewer AI (Multi-Agent System)

Привіт! Це мій проєкт — Telegram-бот, який не просто "чатиться", а реально допомагає розробникам чистити їхній код.

### 🌟 В чому фішка?

На відміну від звичайних ботів, тут працює **Multi-Agent System**. Це означає, що коли ти надсилаєш код, він проходить через двох незалежних "фахівців":

1. **Bug Hunter** — його завдання знайти критичні помилки, ділення на нуль або дірки в безпеці.
2. **Clean Code Architect** — він отримує звіт від першого агента і переписує код так, щоб він був гарним, зрозумілим і за стандартами.

### 🚀 Технології

* **Мова:** Python (aiogram)
* **ШІ-мізки:** ChatGPT (через OpenRouter API)
* **Архітектура:** Асинхронна (бот не "висне", поки ШІ думає)

---

### 🛠 Як запустити це в себе?

**1. Ставимо бібліотеки:**
Відкрий термінал у папці з проєктом і пропиши:

```bash
pip install aiogram openai python-dotenv

```

**2. Налаштовуємо ключі:**
Тобі знадобиться файл `.env` (я додав `.env.example`, щоб ти бачив формат). Просто створи файл `.env` і встав туди свій токен від BotFather та ключ від OpenRouter:

```env
TELEGRAM_TOKEN=твій_токен
OPENROUTER_API_KEY=твій_ключ

```

**3. Стартуємо:**

```bash
python index.py

```
